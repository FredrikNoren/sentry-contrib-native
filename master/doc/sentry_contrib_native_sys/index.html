Description

Unofficial FFI bindings to the Sentry Native SDK for Rust. This crate isn't intended to be used directly, use sentry-contrib-native instead.

For more details see sentry-contrib-native's README

Crate features

  • backend-default - Enabled by default, will use Crashpad on Linux, MacOS and Windows and InProc for Android. See SENTRY_BACKEND at the Sentry Native SDK.
  • transport-default - Enabled by default, will use WinHttp on Windows and Curl everywhere else as the default transport.
  • backend-crashpad - Will use Crashpad. See SENTRY_BACKEND at the Sentry Native SDK.
  • backend-breakpad - Will use Breakpad. See SENTRY_BACKEND at the Sentry Native SDK.
  • backend-inproc - Will use InProc. See SENTRY_BACKEND at the Sentry Native SDK.
  • nightly - Enables full documentation through feature(external_doc).
